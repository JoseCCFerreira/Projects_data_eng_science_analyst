Criei um hub central para apresentar os meus projetos de Engenharia de Dados, Ciência de Dados e Analytics:

**Projects_data_eng_science_analyst**

GitHub:
https://github.com/JoseCCFerreira/Projects_data_eng_science_analyst

Este repositório não é apenas uma coleção de notebooks. É uma camada central de acesso que liga vários projetos end-to-end com:

- dashboards em Streamlit
- SQL e modelação relacional
- referência rápida de PL/SQL
- bases SQLite e DuckDB
- analytics engineering com abordagem dbt
- pipelines de dados
- análise estatística
- machine learning
- documentação visual
- explicações em inglês e português
- scripts para arrancar e testar todos os dashboards

O painel central dá acesso a seis projetos:

1. Global Fuel Shocks Intelligence
2. Citibike E-Bike Performance
3. Portugal Bike Geospatial ML
4. Retail Analytics Learning
5. Retail Case With Beginner HTML
6. Tyre Wear Intelligence

Cada projeto pode correr de forma independente, mas o repositório central funciona como um painel de controlo do portfólio: dashboards, documentação, links dos repositórios, bases de dados, ficheiros, gráficos, estado Git, logs e explicações técnicas ficam acessíveis num só local.

O objetivo foi demonstrar:

- como dados brutos se transformam em outputs analíticos
- como bases relacionais e bases analíticas se complementam
- como SQL, Python, dbt, estatística e ML se ligam em fluxos reais
- como dashboards tornam projetos técnicos mais fáceis de explorar
- como documentação transforma código num produto compreensível
- como SQL, PL/SQL, SQLite e DuckDB entram em contextos diferentes

O repositório inclui diagramas sobre:

- arquitetura central
- portfólio de projetos
- pipeline de dados
- fluxo Git
- conceitos de matemática, estatística e ML
- mapa de acesso aos dashboards

Este sistema foi construído como portfólio prático para funções de Data Analyst, Data Engineer, Analytics Engineer e Data Scientist.
