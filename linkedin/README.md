# LinkedIn Presentation Pack

This folder contains ready-to-use material to present the repository on LinkedIn.

Repository:

https://github.com/JoseCCFerreira/Projects_data_eng_science_analyst

## Files

- `post_en.md` - English LinkedIn post.
- `post_pt.md` - Portuguese LinkedIn post.
- `carousel_script.md` - 10-slide carousel structure.
- `technical_summary.md` - concise technical breakdown for comments, recruiters or portfolio pages.
- `profile_featured_text.md` - short text for the LinkedIn Featured section.

## Suggested Publishing Flow

1. Open the central dashboard locally and take 2-3 screenshots.
2. Use `carousel_script.md` to create a LinkedIn carousel.
3. Publish using `post_en.md` or `post_pt.md`.
4. Add the GitHub repository link in the post comments and Featured section.
