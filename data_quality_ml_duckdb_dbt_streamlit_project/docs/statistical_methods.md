# Statistical Methods

- **Média**: valor médio de uma distribuição.
- **Mediana**: valor central, menos sensível a outliers.
- **Desvio padrão**: dispersão em torno da média.
- **Correlação**: relação entre duas variáveis.
- **Outliers**: valores fora de padrões esperados.
- **Z-score**: distância em desvios padrão.
- **IQR**: intervalo interquartil usado para detectar outliers.
- **Skewness**: assimetria da distribuição.
- **Kurtosis**: cauda e achatamento da distribuição.
- **Resampling**: agrupar dados em novos intervalos de tempo.
- **Filtros de ruído**: suavização para reduzir flutuações indesejadas.
