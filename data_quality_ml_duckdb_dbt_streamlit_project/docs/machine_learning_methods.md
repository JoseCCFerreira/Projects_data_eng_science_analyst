# Machine Learning Methods

## Clustering

- **K-Means**: separa dados em grupos com base em distância.
- **DBSCAN**: identifica clusters densos e ruído.
- **Agglomerative Clustering**: constrói hierarquia de grupos.
- **PCA**: reduz dimensão para visualização e interpretação.

## Modelos preditivos

- **Regressão**: prevê valores numéricos como risco de falha.
- **Classificação**: prevê classes ou estados.
- **Avaliação**: MAE, RMSE, R² para regressão; acurácia, precisão, recall, F1 para classificação.
- **Overfitting**: quando o modelo aprende ruído em vez de padrão.
- **Train/test split**: separa dados para treino e validação.
- **Feature importance**: indica quais variáveis são mais impactantes.
