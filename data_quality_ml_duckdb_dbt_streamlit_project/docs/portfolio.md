# Portfolio Story

## Problema

Dados industriais de sensores são ruidosos, irregulares e apresentam problemas de qualidade que afetam decisões operacionais.

## Solução

Este projeto gera um pipeline completo com ingestão, armazenamento transacional, camada analítica, transformações dbt, análise de qualidade, limpeza, resampling, filtros, clustering, predição e dashboard.

## Tecnologias

Python, SQLite, DuckDB, dbt, Streamlit, Plotly, scikit-learn.

## Impacto

Mostra capacidade de construir pipelines escaláveis, validar dados, extrair padrões e apresentar resultados de forma interativa.

## Próximos passos

Adicionar orquestração, CI/CD, cloud analytics e monitorização de modelos.
