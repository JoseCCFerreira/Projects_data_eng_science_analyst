PL/SQL

Oracle procedural SQL for procedures, functions, triggers, loops and exception handling. Use it when business logic must live close to an Oracle database.

BEGIN
  DBMS_OUTPUT.PUT_LINE('Hello PL/SQL');
END;
/

Git

Version control for code, documentation and project history. Use it after every coherent tested change.

git status --short
git diff
git add .
git commit -m "message"

SQL + SQLite

Relational querying and lightweight local OLTP-style databases for learning tables, keys and joins.

SELECT customer_id, SUM(total)
FROM transactions
GROUP BY customer_id;

DuckDB

Local analytical database for CSV/Parquet, aggregations, marts and dashboard backends.

SELECT * FROM read_csv_auto('data.csv');

dbt

SQL transformation framework for staging, intermediate and mart models with tests and lineage.

dbt debug
dbt build
dbt docs generate

Statistics

Measured evidence: means, variance, z-scores, correlations, distributions and event studies.

z = (x - mean) / std

Machine Learning

Feature-based prediction, classification, clustering, recommendations and model interpretation.

features -> model -> metrics -> limits

Streamlit

Interactive dashboards for charts, filters, tables, downloads and model exploration.

streamlit run app.py --server.port 8610
