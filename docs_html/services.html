How To Start A Dashboard

1

Open the central Streamlit panel at http://localhost:8600.

2

Go to Services and choose the project.

3

Click Install / Update Dependencies if the project environment is not ready.

4

Click Start App, then use the dashboard link.

How Pipelines Fit In

Pipelines refresh generated data, databases and outputs. Dashboards read those outputs. The central manager keeps both actions visible: run the pipeline, inspect the log, open the dashboard, then validate the result.
