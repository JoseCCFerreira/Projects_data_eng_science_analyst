PL/SQL

SQL procedural da Oracle para procedures, functions, triggers, ciclos e tratamento de exceções. Usa quando a lógica deve viver perto da base Oracle.

BEGIN
  DBMS_OUTPUT.PUT_LINE('Hello PL/SQL');
END;
/

Git

Controlo de versões para código, documentação e histórico de projeto. Usa depois de cada alteração coerente e testada.

git status --short
git diff
git add .
git commit -m "message"

SQL + SQLite

Consultas relacionais e base local leve para aprender tabelas, chaves e joins.

SELECT customer_id, SUM(total)
FROM transactions
GROUP BY customer_id;

DuckDB

Base analítica local para CSV/Parquet, agregações, marts e dashboards.

SELECT * FROM read_csv_auto('data.csv');

dbt

Framework de transformação SQL para modelos staging, intermediate e marts com testes e lineage.

dbt debug
dbt build
dbt docs generate

Estatística

Evidência medida: médias, variância, z-scores, correlações, distribuições e estudos de eventos.

z = (x - media) / desvio

Machine Learning

Previsão, classificação, clustering, recomendações e interpretação de modelos com features.

features -> model -> metrics -> limits

Streamlit

Dashboards interativos para gráficos, filtros, tabelas, downloads e exploração de modelos.

streamlit run app.py --server.port 8610
