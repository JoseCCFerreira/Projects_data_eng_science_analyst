Central Streamlit Panel

The central app discovers repositories in projects/, reads documentation, explores CSV/Excel/Parquet/JSON files, opens DuckDB/SQLite databases and starts individual project dashboards.

streamlit run project_manager_app.py --server.port 8600

Repository Model

The root repository versions the access layer, documentation and registry. Each project inside projects/ keeps its own Git history.

End-To-End Learning

Git tracks change, SQLite models relational transactions, DuckDB powers analytics, dbt structures SQL transformations, Python builds analysis and ML, and Streamlit presents the results.

Dashboard Links

Each project has a dedicated local dashboard URL and can be started from the central panel.
