Data Artifacts

CSV, DuckDB and SQLite files are the core artifacts. Inspect rows, columns, types, coverage and missing values before interpreting charts.

Charts

A chart should answer a specific question: trend, distribution, ranking, correlation, geography or comparison between groups.

Interpretation

A result needs context: source, time window, method, metric definition, assumptions and causal limitations.

Result Review Checklist

1

Confirm that the pipeline or validation command passed.

2

Open the generated tables and inspect rows, columns and types.

3

Create a simple chart before trusting a complex model.

4

Read model metrics together with a baseline.

5

Write conclusions with explicit limitations.
